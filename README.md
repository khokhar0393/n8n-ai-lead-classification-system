# AI Lead Classification System

An AI-powered lead qualification workflow built with n8n and OpenAI.

## Features

* Receive leads through Webhook
* AI-powered lead analysis
* Automatic lead categorization
* Priority scoring (High, Medium, Low)
* Lead summary generation
* Sentiment detection
* High-priority email notifications
* Google Sheets integration
* Automated lead routing

## Workflow

Webhook → OpenAI AI Agent → Structured Output Parser → IF Condition → Gmail Alert → Google Sheets

## AI Output

The AI analyzes each lead and returns:

* Category
* Priority
* Summary
* Sentiment
